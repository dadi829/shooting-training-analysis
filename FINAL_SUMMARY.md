## 🚀 七、快速开始指南

### 1. 启动后端服务
```bash
cd d:\web-ai\backend
npm install
node server.js
```
后端将运行在 http://localhost:3002

### 2. 启动前端服务
```bash
cd d:\web-ai
npm install
npm run dev
```
前端将运行在 http://localhost:3001

### 3. 使用流程
1. 打开浏览器访问 http://localhost:3001
2. 先注册一个教练账号
3. 再注册一个学员账号，选择该教练
4. 开始使用各项功能
