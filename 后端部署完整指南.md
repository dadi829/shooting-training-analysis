# 后端部署完整指南

## 一、项目概述
后端是基于 Node.js + Express 的 API 服务，用于处理图片上传和AI分析。

## 二、环境要求

### 2.1 服务器环境
- **操作系统**：Linux（推荐 Ubuntu 20.04+）或 Windows Server
- **Node.js 版本**：16.x 或 18.x（建议 18.x）
- **网络要求**：公网IP，开放3002端口
- **内存**：最低 2GB，建议 4GB+

### 2.2 检查Node.js版本

```bash
# 检查版本
node -v
# 如果没有安装，请安装：
# Ubuntu/Debian:
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs
```

---

## 三、部署步骤

### 3.1 上传代码到服务器
将 `backend/` 文件夹上传到服务器（如 `/var/www/shootai-backend/`）。

### 3.2 安装依赖

```bash
# 进入项目目录
cd /var/www/shootai-backend

# 安装npm依赖
npm install
```

### 3.3 配置环境变量

1. 复制示例配置：
```bash
cp .env.example .env
```

2. 编辑 `.env` 文件：
```bash
nano .env
```

3. 填入以下配置：
```env
# 服务器端口
PORT=3002
NODE_ENV=production

# 豆包API配置（必需！）
DOUBAO_API_KEY=你的豆包API密钥
```

**获取豆包API Key：**
- 访问 https://console.volcengine.com/ark/
- 注册账号并创建应用
- 获取 API Key

### 3.4 手动测试后端

```bash
# 启动服务
node server.js

# 看到以下输出表示成功
Server running on port 3002
```

在浏览器访问：`http://你的服务器IP:3002` 测试。

## 四、配置后台运行（PM2方式，推荐）

### 4.1 安装 PM2

```bash
npm install -g pm2
```

### 4.2 使用 PM2 启动服务

```bash
# 进入项目目录
cd /var/www/shootai-backend

# 启动服务
pm2 start server.js --name shootai-backend

# 查看服务状态
pm2 status

# 查看日志
pm2 logs shootai-backend
```

### 4.3 设置开机自启

```bash
# 保存当前PM2进程
pm2 save

# 生成开机自启脚本
pm2 startup
# 按照提示运行输出的命令
```

### 4.4 常用 PM2 命令

```bash
pm2 restart shootai-backend    # 重启服务
pm2 stop shootai-backend       # 停止服务
pm2 delete shootai-backend     # 删除服务
pm2 logs shootai-backend --lines 100  # 查看最近100条日志
```

## 五、配置防火墙和云服务器

### 5.1 云服务器安全组配置

**阿里云/腾讯云/华为云：**
1. 进入安全组管理
2. 添加入站规则：
   - **协议类型**：TCP
   - **端口**：3002
   - **授权策略**：允许
   - **授权对象**：0.0.0.0/0（或只允许特定IP，更安全）

### 5.2 系统防火墙配置（Ubuntu UFW）

```bash
# 检查防火墙状态
sudo ufw status

# 开放3002端口
sudo ufw allow 3002/tcp

# 重启防火墙
sudo ufw reload
```

## 六、配置 Nginx 反向代理（可选，推荐生产环境）

### 6.1 安装 Nginx

```bash
sudo apt-get update
sudo apt-get install -y nginx
```

### 6.2 配置 Nginx

创建配置文件：
```bash
sudo nano /etc/nginx/sites-available/shootai-backend
```

填入以下内容：
```nginx
server {
    listen 80;
    server_name your-domain.com;  # 你的域名或服务器IP

    location / {
        proxy_pass http://127.0.0.1:3002;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

启用配置：
```bash
sudo ln -s /etc/nginx/sites-available/shootai-backend /etc/nginx/sites-enabled/
sudo nginx -t  # 测试配置
sudo systemctl reload nginx  # 重启Nginx
```

### 6.3 配置 HTTPS（可选，生产环境建议）

使用 Certbot 获取免费证书：
```bash
sudo apt-get install -y certbot python3-certbot-nginx
sudo certbot --nginx
```

## 八、常见问题与解决方案

### 问题1：端口被占用
```bash
# 查看占用3002端口的进程
sudo lsof -i :3002

# 杀掉进程
sudo kill -9 PID
```

### 问题2：无法访问API
- 检查安全组是否开放3002端口
- 检查防火墙是否开放
- 检查服务是否在运行：`pm2 status`
- 查看日志：`pm2 logs shootai-backend`

### 问题3：API Key无效
- 检查 `.env` 文件中的API Key是否正确
- 检查豆包账户是否有额度
- 查看日志：`pm2 logs shootai-backend`
