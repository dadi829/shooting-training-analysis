# 🎯 射击训练分析系统 - 项目总结

## 📋 项目概述

这是一个基于React + TypeScript + Vite + Node.js构建的射击训练数据分析管理系统，支持教练端和学员端分离，提供AI图片分析、历史记录管理、数据可视化等功能。

## 🏗️ 技术架构

## 🚀 快速开始

### 1. 后端启动
```bash
cd backend
npm install
node server.js
```
后端将运行在 `http://localhost:3002`

### 2. 前端启动
```bash
cd ..
npm install
npm run dev
```
前端将运行在 `http://localhost:5173`

### 3. 使用流程
1. 访问前端页面
2. 先注册一个教练账号
3. 再注册学员账号，选择刚才的教练
4. 登录后开始使用

## 📊 核心API接口

### 用户认证
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/login` - 用户登录
- `POST /api/auth/logout` - 用户登出
- `GET /api/auth/me` - 获取当前用户信息
- `GET /api/coaches` - 获取教练列表

### 训练记录
- `GET /api/records` - 获取记录列表（带权限控制）
- `POST /api/records` - 上传新记录（带用户信息）
- `DELETE /api/records/:id` - 删除记录

### 教练专属
- `GET /api/coach/students` - 获取我的学员列表

3. 可以查看所有学员的训练记录

3. 查看自己的历史记录
